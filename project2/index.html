Table tags

HTML Tag

Name

Description

<table>

Table

The wrapper element for all HTML tables.

<thead>

Table Head

The set of rows defining the column headers in a table.

<tbody>

Table Body

The set of rows containing actual table data.

<tr>

Table Row

The table row container.

<td>

Table Data

The table row container.

<tfoot>

Table Foot

The set of rows defining the footer in a table.

What do I need to make my table look professional?

HTML

Required for the table structure, without it, there won't be a table.

CSS

Required for full personalization, color, sizing and more stuff.
